# PDF_ReviewTick
資料や図面PDF内の指定テキストを自動検索し、検図済みの印（●や✔）を一括付与して作業時間を劇的に短縮する、実務直結型の検図補助ツールです
（OCRではありません。テキストデータが含まれるPDFを前提としています。）

## 📌 概要
ReviewTick は、PDF形式の図面や技術資料における検図作業を効率化するためのデスクトップアプリケーションです。
指定したキーワード（図番、定数、部品番号など）を自動で探し出し、確認の印として「●（ドット）」や「✔（チェックマーク）」を正確な位置に書き込みます。

チェックしました感を出す場合　「✔（チェックマーク）」
一文字一文字見ました感を出す場合　「●（ドット）」
を選べます。
（どちらも、チェック内容は同じです。　目視よりは確実です。）

## ✨ 主な機能
一括注釈付与: 複数のテキストを一括検索し、すべての出現箇所に注釈を追加。

注釈スタイルの選択: 各文字の上に打つ「●」と、項目の右端に打つ「✔」を切り替え可能。

範囲指定検索: マウスドラッグで選択した特定の矩形エリア内のみを検索対象に設定。

サムネイルナビゲーション: サイドバーのサムネイルから素早くページを切り替え。

CSVサマリー出力: 実行日時、ページ番号、検出座標を含む処理ログを自動保存。

エラーハンドリング: ファイルが他のアプリで開かれている際の保存失敗を防ぐ通知機能。

## 🚀 使い方
PDFを開く: 対象のPDFファイルを読み込みます。

検索対象を入力: 検索したい語句をテキストエリアに改行区切りで入力します。　
（チェックしたい文字列が資料や図面に書かれていることを自動でチェックします。
ここには確実に正しい文字列を入力してください。
目視の不確実さを補いPDFファイルに✔を打つという作業を自動化します。）

設定の調整: 注釈の色、サイズ、上下のオフセットをスライダーで調整します。

実行: 「全ページを処理」ボタンを押し、保存先を指定します。
